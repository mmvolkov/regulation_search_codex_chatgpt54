# Logging Smoke Test Checklist

Этот файл нужен для короткой проверки после деплоя без чтения полного runbook.

## 1. Авторизация

1. Открыть production UI.
2. Ввести рабочий email.
3. Нажать `Проверить доступ`.

Ожидается:

- успешный ответ в UI
- новая строка в `auth_log`

## 2. Поиск

1. Выполнить один запрос по регламентам.

Ожидается:

- ответ в UI
- в ответе есть `requestId`
- новая строка в `auth_log`
- новая строка в `interaction_log`

## 3. Feedback

1. После ответа нажать `Да` или `Нет`.

Ожидается:

- UI показывает подтверждение сохранения
- новая строка в `feedback_log`
- `request_id` совпадает с запросом из `interaction_log`

## 4. Загрузка документа

1. Загрузить один `.docx`.

Ожидается:

- успешный ответ в UI
- новая строка в `auth_log`
- новая строка в `document_loading_log`

## 5. Очистка коллекции

1. Выполнить очистку коллекции под `admin`.

Ожидается:

- успешный ответ
- новая строка в `document_loading_log`
- `action = collection_clear`
