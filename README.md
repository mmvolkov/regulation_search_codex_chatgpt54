# Regulation Search

Проект для поиска по корпоративным регламентам в формате `docx`.

Исходные документы лежат в каталоге `documents/`.

Планируемая архитектура:

- парсинг `docx` с сохранением структуры разделов и таблиц;
- гибридный поиск по Qdrant;
- backend-логика в `n8n`;
- фронтенд на Beget.
